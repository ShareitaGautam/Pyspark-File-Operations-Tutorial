# PySpark File Operations Tutorial

This repository contains beginner-friendly examples of **PySpark file operations**.

The goal of this project is to demonstrate how to:

- Read CSV files using PySpark
- Use headers and infer schemas
- Define a custom schema
- Write data in different formats
- Run SQL queries on Spark DataFrames

Everything is written with **simple comments so beginners can understand easily.**

---

# Technologies Used

- Python
- Apache Spark (PySpark)
- PyCharm
- Hadoop File System paths
